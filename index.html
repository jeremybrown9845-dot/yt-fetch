Quick Setup
1
Install yt-dlp
pip install yt-dlp  # or: brew install yt-dlp
2
Install ffmpeg (for merging streams)
brew install ffmpeg  # macOS · apt install ffmpeg # Ubuntu
3
Run the companion server (optional, enables live UI)
python3 server.py  # starts API on http://localhost:8765
4
Paste a URL above — or copy the generated command to your terminal
